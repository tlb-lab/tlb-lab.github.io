Conversion#

Utilities for converting Graphein Networks to Geometric Deep Learning formats.

class graphein.ml.conversion.GraphFormatConvertor(src_format: str, dst_format: str, verbose: SUPPORTED_VERBOSITY = 'gnn', columns: Optional[List[str]] = None)[source]#

Provides conversion utilities between NetworkX Graphs and geometric deep learning library destination formats. Currently, we provide support for converstion from nx.Graph to dgl.DGLGraph and pytorch_geometric.Data. Supported conversion formats can be retrieved from SUPPORTED_FORMATS.

Parameters

  • src_format (Literal["nx", "pyg", "dgl"]) – The type of graph you’d like to convert from. Supported formats are available in SUPPORTED_FORMATS

  • dst_format (Literal["nx", "pyg", "dgl"]) – The type of graph format you’d like to convert to. Supported formats are available in: graphein.ml.conversion.SUPPORTED_FORMATS

  • verbose (graphein.ml.conversion.SUPPORTED_VERBOSITY) – Select from "gnn", "default", "all_info" to determine how much information is preserved (features) as some are unsupported by various downstream frameworks

  • columns (List[str], optional) – List of columns in the node features to retain

Diffusion Matrices#

Generate diffusion matrices based on NetworkX graph objects.

Author: Arian Jamasb & Eric Ma

Each of the functions here accepts a NetworkX graph object and returns a 2D xarray. These arrays can then be stacked to generate a diffusion tensor.

Return the matrix power of the adjacency matrix.

Parameters

  • amat_kwargs (Dict[str, Any]) – Keyword arguments to configure NetworkX’s adjacency_matrix function. Defaults to {}

  • with_identity (bool) – Whether or not to add in the identity matrix to the adjacency matrix. Effectively adding “self loops”. Defaults to True.

  • power (float) – Matrix power to raise the adjacency matrix, defaults to 1.

graphein.ml.diffusion.inverse_distance_matrix(G: networkx.classes.graph.Graph, power: float) → xarray.core.dataarray.DataArray[source]#

Return the inverse distance matrix.

Using the coordinates present on the graph object, calculate the inverse power distance matrix.

The resultant array will be symmetric, with entries [i, j] corresponding to 1 / (distance**power), where distance == euclidean distance between the (x, y, z) coordinates of the two graphs. Diagonals (entries [i, i]) are set to 0.

Parameters

  • G (nx.Graph) – NetworkX Graph object. Assumes that each node has x_coord, y_coord, and z_coord as node attributes.

  • power (float) – The power for the distance calculation.

Returns

the inverse distance matrix.

Type

xr.DataArray
